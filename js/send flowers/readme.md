# Js 强大表现力
- 对象由两部分构成
    语法
    -属性
    -方法
- 现实世界使用对象字面量映射到代码世界
- 面向对象不是单纯的面向对象，现实世界的复杂性如何简化
    - 间接送花

- 小美和小红两个对象，因为都有receiveFlower方法，具有同样的接口，实现了相同接口的对象可以互换使用  -->代理模式
- {key:value} 描述对象  这个叫对象字面量
- JS 数据类型分为两大类
    - 基本数据类型
        - number
        - string
        - boolean
    - 引用数据类型（复杂数据类型）
        - object

- typeof 获取数据类型
    typeof tom
    typeof tom.age
    typeof tom.sendFlower -->function  也是一个对象
